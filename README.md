# 🎮 ASRplus — плагин автовыдачи аккаунтов для FunPayCardinal

**ASRplus** — плагин для FunPayCardinal, который полностью автоматизирует аренду Steam-аккаунтов на FunPay.

---

## ✨ Возможности

- 🤖 **Авто-выдача** — при новом заказе аккаунт выдаётся покупателю мгновенно и автоматически
- 🔄 **Авто-продление** — если покупатель оплачивает лот повторно с тем же тегом, аренда продлевается автоматически
- 🔒 **Авто-смена пароля** — после окончания аренды пароль меняется автоматически, аккаунт освобождается
- 🟢 **Авто-вкл/выкл лотов** — лоты на FunPay включаются когда появляются свободные аккаунты и выключаются когда склад пуст
- 🏷️ **Система тегов** — каждый аккаунт и лот привязывается к тегу, плагин сам определяет какой аккаунт выдать
- 📌 **@тег в описании лота** — достаточно написать `@тег` в подробном описании лота на FunPay, плагин подхватит его автоматически
- ⏰ **Предупреждение об окончании** — за 10 минут до конца аренды покупатель получает предупреждение
- 💰 **Авто-возврат** — при ошибке выдачи автоматически делает возврат покупателю
- 📊 **Статистика** — аренды за день/неделю/месяц, топ покупателей и аккаунтов
- 📱 **Telegram-уведомления** — все события приходят в Telegram с кнопками управления

---

## 📦 Установка

1. Скачайте файл `asr_plus.py` из последнего релиза
2. Положите его в папку `plugins/` вашего FunPayCardinal
3. Перезапустите кардинала
4. Откройте Telegram-бота и введите команду `/asrplus`

---

## ⚙️ Настройка

### 1. Добавьте аккаунты
В меню **📂 Аккаунты → ➕ Добавить** введите логин, пароль и загрузите `.mafile`.

### 2. Добавьте лоты
В меню **🔗 Лоты → ➕ Добавить** укажите ID лота с FunPay и привяжите тег.

### 3. Укажите @тег в описании лота на FunPay
В **подробном описании** лота напишите `@тег`, например `@peak`. Плагин будет автоматически определять нужный аккаунт по этому тегу.

### 4. Включите авто-выдачу
На главном экране нажмите кнопку **Авто-выдача**.

---

## 🏷️ Система тегов

Каждый аккаунт имеет тег (например `peak`, `arma3`). Каждый лот тоже привязан к тегу. При заказе плагин определяет тег лота и выдаёт свободный аккаунт с тем же тегом.

## 💬 Команды покупателя в чате FunPay

| Команда | Действие |
|--------|----------|
| `!код` | Получить Steam Guard код |
| `!время` | Узнать сколько времени осталось |
| `!продлить` | Узнать как продлить аренду |
| `!аккаунт` | Получить данные аккаунта повторно |
| `!склад` | Посмотреть наличие |
