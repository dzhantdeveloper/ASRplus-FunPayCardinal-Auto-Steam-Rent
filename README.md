# ASR+ — Auto Steam Rental Plugin for Cardinal

**Версия:** 1.0.0  
**Автор:** [@Dzhants](https://t.me/Dzhants)  
**Платформа:** [Cardinal (FunPay)](https://github.com/sidor0912/FunPayCardinal)

Плагин для автоматической почасовой аренды Steam аккаунтов через FunPay.  
1 час аренды = 1 единица товара в лоте.

## ✨ Возможности

- **Авто-выдача** — при поступлении заказа аккаунт выдаётся покупателю автоматически
- **Steam Guard** — покупатель получает актуальный код по команде `код` / `code`
- **Таймер аренды** — покупатель видит остаток времени командой `время` / `time`
- **Продление аренды** — прямо в чате командой `продлить` / `extend`
- **Автосмена пароля** — после завершения аренды пароль меняется автоматически
- **Авто-возврат** — при ошибке выдачи заказ возвращается покупателю
- **Управление лотами** — автовкл/выкл лотов в зависимости от наличия свободных аккаунтов
- **Бонусы за отзыв** — настраиваемые бонусные часы за оставленный отзыв
- **Чёрный список** — блокировка нежелательных покупателей
- **Bulk-загрузка** — массовая загрузка аккаунтов через `.mafile`
- **Статистика** — выручка, топ тегов, история заказов, экспорт в `.txt`
- **Ручная выдача** — ручной старт аренды из Telegram-бота

## 📦 Установка

1. Скопируйте `asr.py` в папку `plugins/` вашего Cardinal
2. Перезапустите Cardinal
3. Настройте плагин через Telegram-бота командой `/asrplus`

## ⚙️ Требования

- Cardinal (FunPay)
- Python 3.10+
- Зависимости устанавливаются автоматически при первом запуске
