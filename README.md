# ASR Plus — Auto Steam Rent Plus

Расширенный плагин для автоматической аренды Steam аккаунтов на платформе FunPay.  
Основан на Auto Steam Rent от @kewanmov.

**Автор:** @Dzhanto  
**Версия:** 0.1 Beta

---

## Особенности

- **1 штука на FunPay = 1 час аренды** — никаких фиксированных периодов
- **Авто-лоты** — лот выключается когда нет свободных аккаунтов, включается автоматически когда появляется свободный
- **Нет кнопки FREE** — все аккаунты сразу доступны для аренды
- **Команда !code / !extend** — работают только после оплаты заказа, иначе покупатель получает отказ
- **Команда !time** — показывает сколько осталось до конца аренды
- **Команда !stock** — показывает количество свободных аккаунтов
- **Команда /asr** — быстрый вызов главного меню в Telegram
- **Кнопка «О плагине»** — версия, автор, описание возможностей
- **Кнопка «Конфиг»** — настройка авто-лотов и всего плагина
- **Поиск аккаунтов** по тегу, нику, логину
- **Смена пароля** происходит только после полного окончания аренды (с учётом всех продлений)

---

## Установка

1. Скопируй файл `asr_plus.py` в папку `plugins/` твоего Cardinal
2. Перезапусти Cardinal
3. Открой меню через команду `/asr` в Telegram боте
4. Добавь аккаунты и настрой лоты

---

## Команды покупателя

| Команда | Описание |
|--------|----------|
| `!code` | Получить Steam Guard код |
| `!time` | Узнать сколько осталось аренды |
| `!stock` | Узнать наличие свободных аккаунтов |
